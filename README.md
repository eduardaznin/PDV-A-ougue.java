# PDV-A-ougue.java
PDV desenvolvido em Java para um açougue que registrava vendas no caderno. O sistema oferece controle de estoque, cadastro de clientes, emissão de nota fiscal e registro de vendas — digitalizando e organizando a gestão do negócio como solução para um problema do mundo real.

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------AçougueSystem — PDV para Açougue
Projeto final da disciplina de Desenvolvimento de Software, desenvolvido em dupla.
O sistema surgiu de um problema real: um açougue local controlava todas as vendas e clientes em um caderno. O objetivo foi digitalizar essa operação com uma solução feita inteiramente em Java.

Funcionalidades

Login com controle de tentativas e troca de senha
Cadastro de clientes e produtos
PDV — registro de vendas
Controle de estoque
Emissão de nota fiscal
Relatórios de vendas
Consulta, alteração e exclusão de registros


Tecnologias

Java (POO)
Persistência em arquivos .txt / .dat
Terminal interativo (menu via Scanner)


Conceitos de POO Aplicados

Herança e Polimorfismo
Classes abstratas e Interfaces
Encapsulamento (getters/setters)
Composição de classes
Sobrescrita de métodos (@Override)
